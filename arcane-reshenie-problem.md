# 🛠 Решение проблем запуска

### Частые ошибки и их исправление:

1. **Ошибка при запуске лаунчера / Защитник Windows:**
   - Отключите Защитник Windows и сторонние антивирусы (Kaspersky, Avast, Malwarebytes).
   - Убедитесь, что вытянули файлы из архива перед запуском.

2. **Ошибка загрузки драйвера (Driver Error / HVCI):**
   - Отключите «Изоляцию ядра» (Core Isolation / HVCI) в настройках безопасности Windows.
   - Отключите виртуализацию в BIOS (VT-x / AMD-V) при необходимости.

3. **Secure Boot (Безопасная загрузка):**
   - Проверьте требования конкретного чита — для некоторых продуктов требуется выключить Secure Boot в BIOS.
